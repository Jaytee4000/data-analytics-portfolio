# Sales Dashboard Analysis

## Overview
This project analyzes  E-Commerce sales performance, revenue trends, and product insights.

## Data Source
The dataset was sourced from a Kaggle challenge, containing customer and basket transaction details from an e-commerce platform.

## Tools Used
- Excel
- Power Query
- Power Pivot

## Approach
The data model was built using an inner join in Power Query (Customer Details + Basket Details) to focus only on customers with valid and complete transaction records.

## Key Insights & Metrics
- Customer Base: 72 active customers
- Average Age: 35 years
- Total Baskets Processed: 155
- Total Customer Tenure: 4,045 months
- Gender Distribution:
  - Male → 81%
  - Female → 19%
- Top Age Segment: Customers aged 25–34 account for the highest basket activity
- Strongest Sales Days: Monday and Saturday show the highest transaction volumes
- Month Comparison: Sales volume in May significantly outperformed June, suggesting seasonality and more precisely campaign influence  

## Skills Demonstrated
- Data Cleaning and Transformation (Power Query)
- Data Modeling (Inner Join)
- Pivot Table Analysis
- KPI & Trend Visualization
- Dashboard Design and Interactivity using slicers (Gender, Age, Day of Week)

## Insight
 This dashboard provides a quick yet powerful view of customer behavior — showing how gender, age group, and day of week influence e-commerce activity within a specific timeframe
